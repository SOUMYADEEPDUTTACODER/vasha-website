# Eraser.io Prompt - Vasha AI Language Support Chart

## Prompt for Eraser.io

Create a compact, horizontal language support chart for Vasha AI showing Global and Indic languages. The design should be minimal, space-efficient, and suitable for poster presentation.

### Design Requirements:
- **Layout**: Horizontal, side-by-side
- **Background**: White (#FFFFFF)
- **Style**: Minimal, clean, compact
- **Font Size**: 10-11px for languages, 12-13px for headers
- **Spacing**: Minimal padding (10-15px between elements)
- **Width**: Optimized for narrow poster space

### Structure:

**Left Section - Global Languages:**
- Package/Container (Blue #4A90E2):
  - Header: "Global Languages (14)"
  - Shape: Rounded rectangle container
  - Height: ~180-200px total
  - Contains one inner box

- Language List Box (Light Blue #BBDEFB):
  - Text (compact, comma-separated, 3-4 lines):
    - Line 1: English, Spanish, French, Italian,
    - Line 2: Portuguese, German, Russian, Arabic,
    - Line 3: Persian, Indonesian, Japanese,
    - Line 4: Korean, Chinese, Turkish
  - Shape: Rounded rectangle (fills container)
  - Black font color, 8-9px font size
  - Minimal padding (8-10px)

**Right Section - Indic Languages:**
- Package/Container (Red/Pink #FF6B6B):
  - Header: "Indic Languages (22)"
  - Shape: Rounded rectangle container
  - Height: ~180-200px total (matches left)
  - Contains one inner box

- Language List Box (Light Pink #FFB3BA):
  - Text (compact, comma-separated, 4-5 lines with codes):
    - Line 1: Hindi (hi), Bengali (bn), Tamil (ta), Telugu (te),
    - Line 2: Gujarati (gu), Marathi (mr), Kannada (kn), Malayalam (ml),

    - Line 6: Manipuri (mni), Santali (sat)
  - Shape: Rounded rectangle (fills container)
  - Black font color, 8-9px font size
  - Minimal padding (8-10px)

- **Orientation**: Horizontal (left to right, side-by-side)
- **Width**: Maximum 1000-1100px total (compact for poster)

- **Spacing**: 15px gap between left and right sections
- **Alignment**: Top-aligned sections
- **Box Width**: Each section ~480-500px wide
- **Vertical Spacing**: Minimal (5-8px between header and list)
